Online Appointment System for Beauty Salons

Online Appointment System for Beauty Salons is a Web-based Online appointment system for beauty salons in Lingayen, Pangasinan.

The System is exclusive only for the city of Lingayen, Pangasinan. But it is dynamically. There is three login options, Admin, Manager and the Customers. The Admin who runs the whole System can Add and Delete Managers and Salons. The Manager also can register for her/his Salon, can add Staffs and Services. And for the Customers, they can choose the salob, what service and choose anyone of the staffs who would do it for them via appointment.

Technology Used

  • UBOLD Bootstrap Framework
  • PHP
  • mySQL
  • Ajax
  • WampServer
  • 000Webhost

Pangasinan Universities and Collgeges Information APP

Universities and Collgeges Information app is an android application where in Universiteis and Colleges Informations around the Pangasinan.

It's an offline collection of information for finding or looking for a University or College around the province of Pangasinan. It includes all the available courses, academics offer, tuition fee per year, sem and unit. The applciation also has a search and filter option so that the user can easily filter which specific University/College that offers what the users looking for.
